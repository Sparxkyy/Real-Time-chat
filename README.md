# 🧠 Real-Time Chat App 💬 (Django + Channels + WebSocket)

A real-time chat application built using **Django**, **Django Channels**, **WebSockets**, and **Redis**. This project allows users to chat instantly in private rooms with real-time updates.

---

## 🔧 Tech Stack

- **Backend**: Django, Django Channels
- **Real-Time Communication**: WebSockets
- **Channel Layer**: Redis
- **Frontend**: HTML, CSS (basic responsive UI)
- **Server Interface**: ASGI (via Daphne or Uvicorn)

---

## 📦 Features

- 🔐 User authentication (login/signup)
- 💬 Real-time messaging with WebSockets
- 🗂️ Private chat rooms per user pair
- 🔁 Persistent message history
- 🚦 Live WebSocket connection feedback
